My Projects

Web Responsive

Implementing a responsive framework was paramount to ensure that users could seamlessly navigate the website regardless of the device they were using. Utilizing CSS media queries and flexible grid layouts, I optimized the website for various screen sizes. This approach not only enhanced user experience but also contributed to improved search engine rankings, as search engines increasingly prioritize mobile-friendly websites.


Web Design using Figma

Navigating Figma's intuitive interface, I translated culinary passion into a user-friendly layout. From the enticing menu section to an easy-to-use reservation system, every pixel was placed with the diner's experience in mind. Figma's collaborative features allowed seamless communication with the restaurant owners, ensuring the design aligned with their vision.


Web Design using Figma

I built my fully functional portfolio using HTML, CSS, and Bootstrap, prioritizing simplicity and efficiency. HTML and CSS formed the foundation for a clean and intuitive interface, while Bootstrap's pre-built components and responsive grid system streamlined the design process. This approach not only facilitated mobile-friendly development but also maintained visual consistency across different screen sizes. Meticulous customization of Bootstrap elements and a personal touch harmonized professional aesthetics with a user-centric experience, showcasing the versatility of these foundational web technologies.
