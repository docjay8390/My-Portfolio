# Bernie Fernando Portfolio

A polished one-page portfolio for Bernie S. Fernando, rebuilt for IT HR reviewers and technical interviewers.

## Highlights

- Minimal professional visual system with light and dark modes
- Tailwind CSS 4 styling on a vinext/Next foundation
- Mobile navigation, theme persistence, certificate tabs, and copyable interview brief
- Recruiter-friendly sections for proof, skills, experience, services, projects, certificates, and a first-90-days plan
- Downloadable resume summary at `public/Bernie-Fernando-Resume.md`

## Requirements

- Node.js `>=22.13.0`

## Commands

```bash
npm run dev
npm run build
npm test
```

This project uses Sites-compatible vinext output and keeps local Cloudflare tool state project-local through `vite.config.ts`.
